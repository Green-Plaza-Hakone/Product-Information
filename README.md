# 画像アップロードとURLについて

このリポジトリには、売店で使用する**商品案内用の画像**を追加することができます。

## 画像の追加方法
1. このリポジトリに画像ファイルをアップロードしてください。
2. 画像が追加されると、自動的にURLが作成されます。

## 画像のURLについて
アップロードした画像は、以下の形式のURLで表示されます。

https://green-plaza-hakone.github.io/Product-Information/ファイル名

※「ファイル名」の部分には、**画像ファイルの名前（拡張子を含む）**が入ります。

### 例
https://green-plaza-hakone.github.io/Product-Information/MineralQ10WaterCreamPOP.jpg


このURLをQRコードに設定すると、  
お客様の端末で**広告やアプリ案内なしに、画像のみが表示されます**。

## 注意事項
- ファイル名は、できるだけ **半角英数字** を使用してください。
- 画像形式は `.jpg` または `.png` を推奨します。
- 同じ名前のファイルをアップロードすると、**上書き**されますのでご注意ください。
